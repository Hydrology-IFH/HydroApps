{% trans "Method" %}

{% trans "Starkregenindex (SRI)" %}

{% trans "Der vorliegende Starkregenindex (SRI) ordnet verschiedenste Niederschlagshöhen einer 12-stufigen Starkregenskala zu. Der Starkregenindex wurde dabei ortsspezifisch für 12 verschiedene Dauerstufen erstellt. Je nach Dauerstufe führt eine gegebene Niederschlagsmenge zu einem unterschiedlichen Starkregenindex." %}

{% trans "Die Einteilung von Niederschlagshöhen zu einzelnen Starkregenkategorien erfolgt dabei nach der Methode von" %} Schmitt et al. (2018). {% trans "Für die Starkregenkategorien von 1 bis 7 werden hierfür Niederschläge einer bestimmten Wiederkehrwahrscheinlichkeit herangezogen. Die Niederschlagsmengen der Starkregenkategorien von 8 bis 12 werden über einen, für alle Dauerstufen konstanten, Erhöhungsfaktor aus den Niederschlagshöhen der Starkregenkategorie 7 abgeleitet." %}

{% trans "Generell gilt, dass Niederschlagsereignisse der Starkregenkategorie 1 und 2 zu "normalen" Starkregenereignissen gezählt werden können; Ereignisse der Kategorie 3 bis 5 zählen zu intensiven Starkregenereignissen; Ereignisse der Kategorie 6 und 7 zählen zu den außergewöhnlichen Starkregenereignissen und Ereignisse der Kategorie 8 und mehr können als extreme Starkregenereignisse kategorisiert werden." %}

{% trans "Datengrundlage SRI" %}

{% trans "Der Zuordnung von Niederschlagshöhen zu einer Starkregenskala basiert auf flächenhaft vorliegenden Bemessungsniederschlägen, jeweils für jede der betrachteten Dauerstufen. In diesem Fall wurden als Basisdaten die Bemessungsniederschläge aus" %} KOSTRA-DWD-2020 {% trans "verwendet." %} {% trans "Da die KOSTRA-DWD-2020 Daten aber nicht für jede der im SRI betrachteten Dauerstufen bzw. benötigten Wiederkehrintervalle vorliegen, wurden die entsprechenden Werte über die verfügbaren KOSTRA-Extremwert & Korrektur-Parameter auf dem ordinären KOSTRA-DWD-2020 Raster abgeleitet und anschließend konservativ auf das räumlich höher aufgelöste SRI-Raster (~ 2km x 2km) aufgeteilt." %}

{% trans "Ereignisdaten" %}

{% trans "Die in dieser Web-Anwendung angeführten Starkregenereignisse wurden aus den Daten der " %}{% trans "Radarklimatologie" %} (RADKLIM) {% trans "des Deutschen Wetterdienst (DWD) extrahiert." %} {% trans "Diese Daten sind deutschlandweit in einem Raster von ca. 1km x 1km verfügbar. Für die Auflistung wurden alle Ereignisse berücksichtigt, die im Zeitraum von 2001 bis 2021 auftraten. Mehr Informationen zu dem RADKLIM-Produkt des DWD erhalten Sie" %} {% trans "hier" %}

{% trans "Um die systematische Untererfassung von Starkregenereignissen in den Radardaten im Vergleich zu Stationsmessungen auszugleichen, wurden die Ereignisse mittels eines statistischen Verfahrens korrigiert. Dieses Verfahren ist in" %} Hänsler & Weiler (2020) {% trans "beschrieben." %}

{% trans "Um die RADKLIM-Daten mit den gröberen SRI-Kacheln (ca. 2km x 2km) zu verschneiden, wurden die originalen 5-Minutendaten aus den Rasterpunkten, die sich mit der jeweiligen SRI-Kachel schneiden, entsprechend ihrem Flächenanteil gemittelt." %}

{% trans "Generell werden nur Ereignisse dargestellt, die einem SRI ≥ 1 entsprechen. Weiterhin wurde die maximale Anzahl von dargestellten Ereignissen auf 15 begrenzt. Wurden im betrachteten Zeitraum weniger als 15 Ereignisse mit einem SRI ≥ 1 beobachtet, werden entsprechend weniger Ereignisse dargestellt." %}

{% trans "Um zu verhindern, dass ein Großteil der aufgeführten Ereignisse auf das gleiche Niederschlagsereignis zurückgehen (gleiches Ereignis, aber andere Dauerstufe) wurde zusätzlich noch das Kriterium eingeführt, dass pro Ereignis maximal 4 Dauerstufen in der Tabelle dargestellt werden." %}
